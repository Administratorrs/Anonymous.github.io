Hosts.allow、Hosts.deny 文件

发表于2021-02-09|更新于2021-02-10|网络应用竞赛
|字数总计:202|阅读时长:1分钟|阅读量:

文件优先级： 先 deny，后 allow，如有冲突则按 allow 文件内的规则处理

hosts.allow 文件

hosts.allow 文件内的内容为允许服务列表
默认路径：/etc/hosts.allow
格式：服务名 + IP + allow

hosts.deny 文件

hosts.deny 文件内的内容为拒绝服务列表
默认路径：/etc/hosts.deny
格式：服务名 + IP + deny
